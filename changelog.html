Changelog

What's New.

Keep track of the latest updates and improvements to QuickCopy.

v2.1.0 (Security & Power Update)

Released on March 17, 2026 (16:30 PM)

  • Security Hardening: Implemented XSS patches for HTML attributes and integrated DOMPurify for secure Markdown rendering.
  • Public Snippet Sharing: Generate unique, view-only links for any snippet with one click.
  • PWA Support: QuickCopy is now installable! Added Service Worker for offline caching and a professional manifest.
  • Archiving & Trash: Move snippets to Archive or Trash to keep your main workspace organized.
  • Keyboard Shortcuts: Added Ctrl+Enter to save, Ctrl+F to search, and Alt+1-9 for instant copying.
  • Markdown Support: Toggle M↓ to render beautiful Markdown notes and code blocks.
v2.0.0 (The UI Overhaul)

Released on March 17, 2026 (12:00 PM)

  • Dynamic Masonry Layout: Snippets now stack naturally based on their content height.
  • Responsive Columns: Strictly locked to 1 column on mobile and 3 columns on desktop.
  • Premium Glassmorphism UI: Complete visual overhaul with enhanced transparency and blur effects.
  • Advanced Search: Real-time snippet filtering.
  • Pinning System: Highlight your most important clips.
v1.5.0

Released on March 10, 2026

  • Dark/Light Mode: Initial support for dynamic themes.
  • Google Sign-In: Secure authentication via Google accounts.
  • Profile Dashboard: Dedicated space for user settings.

More exciting updates coming soon! 🚀
