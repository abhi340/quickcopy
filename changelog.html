QuickCopy Pro v2.5.0

The biggest update in QuickCopy history. We've transformed from a simple clipboard into a professional productivity ecosystem.

v2.5.0 (Market-Ready Ecosystem)

Released on March 17, 2026 (17:00 PM)

  • Chrome Extension Foundation: Added `/extension` folder with Manifest V3. You can now load QuickCopy as a browser extension.
  • Professional Toolkit: Generate personal API keys in Settings for secure developer access.
  • Data Portability: Added full JSON Export feature to back up your clips locally.
  • Vim-style Navigation: Navigate your workspace with `j`, `k`, `y`, and `/`.
  • Search Highlighting: Real-time visual feedback while searching through clips.
  • Smart Link Previews: Automatic Favicon fetching and rich card rendering for URLs.
  • Landing Page Overhaul: A stunning new Glassmorphism product page for new users.
v2.1.0 (Cloud & PWA Update)

Released on March 17, 2026 (14:00 PM)

  • Cloudflare Integration: Migrated hosting to Cloudflare Pages for global speed and reliability.
  • Smart Tags: Automatic `#Link` and `#Code` categorization.
  • Public Sharing: Generate unique, view-only links for any snippet.
  • PWA Support: QuickCopy is now installable on Mobile and Desktop.
  • Archiving & Trash: Advanced lifecycle management for your clips.

The roadmap to the Play Store is officially underway! 🚀
